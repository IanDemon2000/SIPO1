1. Kaj je OS

Je programska oprema v računalniku ki deluje kot vmestnik med uporabnikom in strojno opremo. OS tudi upravlja vire računalčnika kor npr. RAM, procesor, datoteke... OS se tudi deli na več slojev. Prvi sloj je strojna oprema, ki je fizični del računalnika ki izvaja vse operacije. Potem gonilniki ki so programi ki omogočajo komunikacijo med OS in strojno opremo. Potem jedro OS ki je srce OS, ki opravlja s procesorjem in spominom. Fizični sloj je kako se podatki fizično zapisujejo na disk. Logočni sloj pa je kako mi vidimo to datoteko na računalniku. Zadnji sloj pa je lupina ali UI, ki uporabnik daja okaze. Obstajata 2 vemstnika, GUI in CLI. GUI ali Graphic user interface, ki lahko uporbnik vidi ikone, okna, svojo miško... Je prijazna uporabniku, samo uporablja več virov. CLI ali command line interface, pa je kot CMD, ki vpisuješ vse ukaze, ki ti ne vidiš kaj se dogaja v ozadju.
