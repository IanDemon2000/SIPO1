1. Kaj je OS

Je programska oprema v računalniku ki deluje kot vmestnik med uporabnikom in strojno opremo. OS tudi upravlja vire računalčnika kor npr. RAM, procesor, datoteke... OS se tudi deli na več slojev. Prvi sloj je strojna oprema, ki je fizični del računalnika ki izvaja vse operacije. Potem gonilniki ki so programi ki omogočajo komunikacijo med OS in strojno opremo. Potem jedro OS ki je srce OS, ki opravlja s procesorjem in spominom. Fizični sloj je kako se podatki fizično zapisujejo na disk. Logočni sloj pa je kako mi vidimo to datoteko na računalniku. Zadnji sloj pa je lupina ali UI, ki uporabnik daja okaze. Obstajata 2 vemstnika, GUI in CLI. GUI ali Graphic user interface, ki lahko uporbnik vidi ikone, okna, svojo miško... Je prijazna uporabniku, samo uporablja več virov. CLI ali command line interface, pa je kot CMD, ki vpisuješ vse ukaze, ki ti ne vidiš kaj se dogaja v ozadju.

3. Win/Linux/macOS- razlika

Prva razlika je cena. Windows je za plačati da dobiš ključ, macOS je vključen v ceno Mac naprave, Linux pa je Brezplačen. Windows je boljši za programe in igre, ker ima več podpore z vsemi programi in igrami. Linux je dober za to, ker ima tudi podporo samo ne toliko kot Windows, zato pa je dobro uporabljati Proton da lahko prevede te windows datoteke v neakaj kaj lahko Linux razume. MacOS pa ima zelo omejeno izbiro, ker je relativno nov in nima veliko podpore. Naslednja stvar je varnost. Windows ima vredu varnost, a je še vedno priporočeno da imaš antivirus ker windows defender včasih preskoči ene stvari ki so lahko nevarne. Za Linux in macOS pa pa imata veliko varnosti in težko vdiranje. Windows je primeren za vsakogar, ker je dober v vsem in se lahko tudi hitro prilagodiš. Linux je bolj priporočen programerjem, ker boš včashi rabil kaj vpisati v ukazno vrstico. MacOS pa je priporočen za ljudje ki radi ustvarjajo, ker ima apple veliko izbiro produktov in je oprimiziran za produktivnost.

4. Kaj pomeni da se naloži ob zagonu

To pomeni da se OS ob zaguonu naloži v RAM, ki lahko računalnik z njim dejansko upravlja. To se zgodi tako da ko računalnik dobi elektriko, se zažene BIOS ali pa UEFI in njegova naloga je da preveri če strojna oprema deluje pravilno, temu procesu se reče POST(Power On Self Test). Potem BIOS/UEFI pogleda tvoj disk, to je lahko ali SSD ali pa HDD in poišče zagonski nalagalnik (bootloader). Ko najde to potem vzame jedro OS in ga iz diska premakne v RAM. Potem OS prevzame nadzor nad računalnikom, ko ima to naloži gonilnike za vse naprave in zažene grafični vmestnik, da lahko vidiš kaj počneš. OS pa se naloži v RAM ker je veliko hitrejši kot če bi bil samo v SSD-ju shranjen.
