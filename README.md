1. Kaj je OS

Je programska oprema v računalniku ki deluje kot vmestnik med uporabnikom in strojno opremo. OS tudi upravlja vire računalčnika kor npr. RAM, procesor, datoteke... OS se tudi deli na več slojev. Prvi sloj je strojna oprema, ki je fizični del računalnika ki izvaja vse operacije. Potem gonilniki ki so programi ki omogočajo komunikacijo med OS in strojno opremo.
